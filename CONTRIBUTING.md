# Contributing Guidelines

Thank you for your interest in this project!

## Important: Licensing Restrictions

**This is NOT an open-source project.** Due to the restrictive custom license, contributions are **very limited**:

- ❌ You **cannot** fork this repository and create your own version
- ❌ You **cannot** submit pull requests to add features or modify code
- ❌ You **cannot** redistribute modified versions of this software
- ✅ You **can** report bugs and security issues (see SECURITY.md)
- ✅ You **can** suggest improvements via GitHub Issues (no guarantee they'll be implemented)

## Reporting Issues

If you find a bug or have a feature request:

1. Check existing GitHub Issues to avoid duplicates
2. Open a new Issue with a clear description
3. Include steps to reproduce (for bugs)
4. Be respectful and constructive

**Note:** Not all issues will be addressed, as this is a personal project maintained by the Owner.

## Bug Reports

When reporting a bug, please include:

- Your operating system and browser (if applicable)
- Steps to reproduce the issue
- Expected vs. actual behavior
- Screenshots or error messages (if relevant)
- Your DAW version (if applicable)

## Feature Requests

Feature requests are welcome, but please understand:

- The Owner prioritizes their own roadmap
- Requests may not be implemented
- This is not a collaborative open-source project

## Code of Conduct

- Be respectful and professional
- No harassment, discrimination, or abusive language
- No spam or self-promotion
- Constructive criticism only
