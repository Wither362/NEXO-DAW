Documentación API v1.0.0 para JS Plugins:
El sistema te inyecta 5 variables automáticamente:
- ctx: AudioContext actual (Úsalo para crear nodos ctx.createOscillator())
- freq: Frecuencia en Hz para la nota del piano roll.
- t: Tiempo de inicio exacto (AudioContext time).
- dur: Duración calculada en segundos según el tempo y los bloques dibujados.
- gainNode: IMPORTANTE. Debes conectar tu último efecto/oscilador aquí (ej: osc.connect(gainNode)) para que el volumen y el EQ del DAW funcionen.
No olvides hacer tuOscilador.start(t) y tuOscilador.stop(t + dur).
